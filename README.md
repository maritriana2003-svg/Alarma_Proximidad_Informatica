# Alarma_Proximidad_Informatica
Proyecto de Alarma de Proximidad para la asignatura Hardware Digital (Prof. Diego Miranda). Implementación física con Arduino Uno (ATmega328P) y sensor HC-SR04. Desarrollado por: Maria Triana, Camila Montes, Fernanda Godoy y Gabriela Ovalle. Facultad de Ingeniería, 2026.
