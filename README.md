# Alarma_Proximidad_Informatica
Proyecto de Alarma de Proximidad para la asignatura Hardware Digital (Prof. Diego Miranda). Implementación física con Arduino Uno (ATmega328P) y sensor HC-SR04. Desarrollado por: Maria Triana, Camila Montes, Fernanda Godoy y Gabriela Ovalle. Facultad de Ingeniería, 2026.
## 👥 IntegrantesMaria Triana   Camila Montes   Fernanda Godoy   Gabriela Ovalle   ## 🔌 Conexiones de Hardware (Pinout)Sensor HC-SR04 (Trigger): Pin 9   Sensor HC-SR04 (Echo): Pin 8   Buzzer (Alarma): Pin 10   LED de Alerta: Pin 11   Pulsador de Control: Pin 7   ## ⚠️ Estado del Proyecto y Desafíos Técnicos Actualmente, el sistema presenta una limitación en la salida sonora (Buzzer). Durante las pruebas físicas, se detectó que el tono se emite con baja intensidad o de forma errática[cite: 26, 27].
